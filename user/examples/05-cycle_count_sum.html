5. CycleCount sum#

a. List of signals (timeseries)#

Note

In this example we define a list of segmented timeseries and compare the sum of the CycleCount from segmented data with the CycleCount from concatenated data.

We then apply the solve_lffd() method, as per LFFD to recover the effect of low-frequency cycles.
